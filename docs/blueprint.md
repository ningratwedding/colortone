# **App Name**: FilterForge

## Core Features:

- Autentikasi Pengguna: Pendaftaran/login aman melalui email.
- Manajemen Profil: Pengguna dapat mengedit profil, avatar, dan bio.
- Eksplorasi Preset & LUT: Jelajahi produk menggunakan tampilan grid/daftar, filter (perangkat lunak), dan pencarian (tag/kategori).
- Pratinjau Produk: Lihat pratinjau before-and-after dari preset/LUT.
- Transaksi Aman: Keranjang belanja, pembayaran aman, dan unduhan otomatis setelah pembelian.
- Favorit & Ulasan: Simpan preset/LUT ke favorit, beri peringkat dan komentar pada mereka.
- Dasbor Penjual: Kreator dapat mengunggah produk, melacak statistik, mengedit/menghapus item.

## Style Guidelines:

- Warna primer: Violet tua (#673AB7) untuk nuansa kreatif dan canggih.
- Warna latar belakang: Abu-abu muda (#ECEFF1) memberikan latar belakang netral untuk menampilkan filter.
- Warna aksen: Cyan (#00BCD4) untuk menyorot elemen interaktif dan ajakan bertindak.
- Font tubuh dan tajuk: 'Inter', sans-serif serbaguna untuk tampilan modern dan netral.
- Ikon bersih dan minimalis untuk mewakili perangkat lunak dan kategori yang berbeda.
- Tata letak berbasis grid untuk tampilan produk untuk memastikan tampilan yang bersih dan terorganisir.
- Transisi yang halus dan animasi halus untuk meningkatkan pengalaman pengguna saat menjelajah atau membeli.
